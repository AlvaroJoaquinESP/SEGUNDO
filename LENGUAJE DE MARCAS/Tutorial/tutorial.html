Tutorial Básico

-- Vamos a repasar el código del vídeo --

Observamos como definimos la variable $aprobado y $nota.

$aprobado = 'Sacaré';
$nota = 10;
echo $aprobado. ' un ' . $nota;

Como podemos observar en el vídeo, este código nos muestra lo siguiente:
Sacaré un 10.


Para dar más énfasis en el mensaje a transmitir hacia el profesor que está corrigiendo este tutorial, el valor de la variable $aprobado ha sido modificada, pasando de true a Sacaré .
