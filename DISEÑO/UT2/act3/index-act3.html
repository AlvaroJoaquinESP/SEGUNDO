Lenguajes de programación

HTML es el lenguaje Frontend más utilizado, y se trata de un lenguaje de marcado que sirve para etiquetar y estructurar una página web.

CSS es una hoja de estilo que se usa para definir el aspecto visual que va a tener la estructura creada en HTML. Son documentos independientes pero que se relacionan e interactúan entre sí.

JavaScript es un lenguaje de programación del lado del cliente, esto es, es un lenguaje de Front-End. Su misión es la de mejorar la interacción de los usuarios con una página web, mejorar su experiencia durante la navegación.

Python fue creado en 1991 por Guido van Rossum, y ha crecido hasta convertirse en uno de los lenguajes de programación multipropósito líderes en el mundo actual. Los desarrolladores de backend aprovechan sus códigos nítidos y altamente legibles para crear scripts funcionales para manejar las asignaciones de backend.

PHP fue desarrollado por Rasmus Lerdorf en 1994. Desde entonces, PHP se ha convertido en uno de los lenguajes de programación del lado del servidor líderes en el mundo. De hecho, una encuesta de W3Tech muestra que más del 79% de los sitios web a nivel mundial funcionan con este lenguaje.

Windows PowerShell es una shell que proporciona acceso a un número mayor y más profundo de funciones y componentes del sistema, y son los preferidos por muchos profesionales de TI y administradores de sistemas.
